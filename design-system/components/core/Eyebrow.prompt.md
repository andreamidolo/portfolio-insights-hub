**Letterspaced ALL-CAPS eyebrow / label.** Sits above section headings and titles feature blocks. Rose by default; navy or white on dark/photo backgrounds.

```jsx
<Eyebrow>What We Do</Eyebrow>
<Eyebrow tone="navy">SEC Registered</Eyebrow>
```

Tones: `rose | navy | muted | white`. Set `as="h3"` to use it as a semantic feature title.
