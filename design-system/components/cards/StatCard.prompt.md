**Serif facts-&-figures statistic.** Big italic-friendly serif value with a caps label — for AUM, client counts, year founded.

```jsx
<StatCard value="2.5" suffix="bn" label="USD assets under management" />
<StatCard value="2009" label="Year established" tone="rose" />
```

`tone="white"` for navy/photo sections; `align="center"` in grids.
