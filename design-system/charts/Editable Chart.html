Editable chart

Every chart in the LFA deck templates is an <lfa-chart>. Hover any chart below and click Edit to open the Chart Studio popup — change the type, edit the data table, paste from Excel, retitle. Changes apply live and persist per chart-id.

Usage: <lfa-chart chart-id="my-id" theme="lfa"></lfa-chart> with a spec registered on window.LFA_CHART_SPECS["my-id"]. Set window.LFA_CHART_EDIT = false to hide the pencil for final exports. The same engine drives the standalone Chart Studio.
